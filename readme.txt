== Description ==

**WP Google Maps Shortcode** - Insert Google Maps into your post/page using Shortcode.

Maps are displayed with the [wp_gmaps] short code:

`[wp_gmaps address="San Francisco, California" zoom="7"]`

== Installation ==

1. Activate the plugin
2. Added [wp_gmaps address="your address here"] or [wp_gmaps lat="your latitude" lng="your longitude"] to any post or page
